# コーディングルール

このプロジェクトにおけるコーディングルールを定めます。コードの一貫性を保ち、可読性とメンテナンス性を向上させることを目的とします。

## 1. 全般

- **言語**: [使用するプログラミング言語]
- **スタイルガイド**: [準拠するスタイルガイド（例: PEP 8, Google Java Style Guide）]
- **文字コード**: UTF-8
- **改行コード**: LF

## 2. 命名規則

- **変数**: `snake_case`
- **定数**: `UPPER_SNAKE_CASE`
- **関数・メソッド**: `snake_case`
- **クラス**: `PascalCase`

## 3. コメント

- 複雑なロジックや、意図が分かりにくい箇所には必ずコメントを追加してください。
- `TODO:` や `FIXME:` などのプレフィックスを活用して、将来のタスクや修正箇所を明示してください。
